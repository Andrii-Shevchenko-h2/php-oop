# What are the "modern" classes
- These classes use the new php 8.4 features
- Ancient are the classes that wouldn't be able to run under 8.4

# What are the classes inside "geometry"
- These classes are used for automatic geometrical shape creation
- They will have publically available privately set properties
- i.e.: area, length, perimeter, radius etc.
- I use this to train php inheritance
